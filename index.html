1. Algoritma Mesin ATM

Kondisi Awal Stok ATM:

  • Rp 100.000 ‚Üí 7 lembar
  • Rp 10.000 ‚Üí 8 lembar
  • Rp 5.000 ‚Üí 8 lembar
  • Rp 1.000 ‚Üí 10 lembar

Prinsip Algoritma:
Mengeluarkan uang dari pecahan terbesar ke terkecil, memeriksa stok yang tersedia, dan jika stok tidak mencukupi, mengeluarkan sisa pecahan yang tersedia.

// Deklarasi variabel global
stok[100000] = 7
stok[10000] = 8
stok[5000] = 8
stok[1000] = 10

PROSEDUR AmbilUang(jumlah)
BEGIN
    pecahan = [100000, 10000, 5000, 1000]

    UNTUK i = 0 SAMPAI panjang(pecahan) - 1 LAKUKAN
        JIKA jumlah >= pecahan[i] DAN stok[pecahan[i]] > 0 MAKA
            ambil = jumlah DIV pecahan[i]

            JIKA ambil > stok[pecahan[i]] MAKA
                ambil = stok[pecahan[i]]
            AKHIR JIKA

            stok[pecahan[i]] = stok[pecahan[i]] - ambil
            jumlah = jumlah - (ambil * pecahan[i])

            CETAK ambil, "lembar Rp", pecahan[i]
        AKHIR JIKA
    AKHIR UNTUK

    JIKA jumlah > 0 MAKA
        CETAK "Stok tidak mencukupi untuk jumlah ini"
    AKHIR JIKA
END

Contoh Eksekusi:
Input: Rp 325.000
Output: 3 lembar Rp 100.000, 2 lembar Rp 10.000, 1 lembar Rp 5.000
